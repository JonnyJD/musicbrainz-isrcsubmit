This python script extracts ISRCs from audio cds and submits them to MusicBrainz.

The script works for Linux, Windows and Mac OS X. However, drutils, the backend for Mac WILL take a VERY long time (several minutes) per disc and might give duplicates and missing ISRCs often. Isrcsubmit will tell you about it though.

Features (as of version {{ site.script.version }})

  • read ISRCs from disc
  • search for releases with the TOC of the disc
  • display release information from MusicBrainz
  • submit (unknown) discIds / TOCs
  • duplicate ISRC detection (local and on server)

Dependencies

  • Python2 >= 2.4
  • python-musicbrainz2 to access the MusicBrainz XMP API version 1
  • libdiscid to calculate an ID for a disc
  • (ctypes with Python < 2.5)
  • one of the usable backends

Usable Backends

  • mediatools
  • Cdrdao
  • cd-info (libcdio)
  • Cdda2wav (in cdrtools)
  • Icedax (in cdrkit)
  • drutil (in Mac OS X; very slow!)

Usage

Usage: isrcsubmit.py [options] [user] [device]
 isrcsubmit.py -h
